Instrucciones para Añadir Imágenes al Sitio Web

Este archivo existe porque las imágenes necesarias para el sitio web no se han agregado todavía. Por favor, sigue las instrucciones a continuación para completar el sitio web.

Imágenes Requeridas

Para que el sitio web funcione correctamente, necesitas añadir las siguientes imágenes:

  • about-image.jpg - Para la sección "Nuestra Historia" (tamaño recomendado: 800x600px)
  • project-1.jpg - Para el primer proyecto (tamaño recomendado: 600x400px)
  • project-2.jpg - Para el segundo proyecto (tamaño recomendado: 600x400px)
  • project-3.jpg - Para el tercer proyecto (tamaño recomendado: 600x400px)

Pasos para Añadir las Imágenes

  1. Prepara tus imágenes con los nombres exactos mencionados arriba.
  2. Asegúrate de que estén optimizadas para web (formato JPG o WebP, tamaño razonable).
  3. Coloca las imágenes directamente en la carpeta img/ del proyecto.
  4. Elimina este archivo placeholder.html después de añadir tus imágenes.

Opciones para Obtener Imágenes

Si no tienes imágenes propias, puedes usar:

  • Imágenes de stock gratuitas de sitios como Unsplash, Pexels o Pixabay.
  • Imágenes generadas por IA con herramientas como DALL-E, Midjourney o Stable Diffusion.

Para instrucciones más detalladas, consulta el archivo INSTRUCCIONES.md en la raíz del proyecto.
